Github Collaboration

Step 1

Fork the repository to your own Github account

Step 2

Clone the forked repository to your local machine

Step 3

Make changes and commit them to your local repository

Step 4

Push the changes to your forked repository on Github

Step 5

Create a pull request to merge your changes with the original repository

Step 6

Wait for the owner of the original repository to review and accept your changes
